## Introduction to XR: Definition

time_limit: 30

XR (Extended Reality) is an umbrella term that encompasses multiple immersive technologies.

**Which of the following best describes XR?**

A. A specific type of VR headset
B. An umbrella term covering VR, AR, and MR
C. A programming language for 3D graphics
D. A networking protocol for real-time communication

> Correct Answer: B. An umbrella term covering VR, AR, and MR
> Overall Feedback: XR (Extended Reality) is an umbrella term that covers Virtual Reality (VR), Augmented Reality (AR), and Mixed Reality (MR). It refers to the entire spectrum of real-to-virtual environments.

## Introduction to XR: Code Example

Consider the following code:

```typescript
const xr = await scene.createDefaultXRExperienceAsync({
    sessionMode: "immersive-vr",
});
```

**What does this code do?**

A. Creates a default scene
B. Initializes VR components
C. Renders a frame
D. Loads a 3D model

> Correct Answer: B. Initializes VR components
> Overall Feedback: The createDefaultXRExperienceAsync method sets up stereo rendering, input handling, and other VR defaults.

## Displays: Multi-Select

time_limit: 45

**Which of the following are types of XR displays? (Select all that apply)**

A. Head-Mounted Display (HMD)
B. CAVE system
C. Blockchain node
D. Holographic display

> Correct Answers: A, B, D
> Overall Feedback: HMDs, CAVE systems, and holographic displays are all types of XR display systems. A blockchain node is a distributed computing concept unrelated to XR displays.
