## Operations: Queue Priority

This synthetic sample mixes plain and code-style prompts.

**A kitchen robot receives orders in a queue. What does FIFO mean?**

A. Fastest item first only
B. First in, first out
C. First in, favorite out
D. Food items only

> Correct Answer: B. First in, first out
> Overall Feedback: FIFO processes orders in arrival order, which keeps handling predictable.

## Code Trace: Python Loop Output

time_limit: 45

Consider this Python code:

```python
total = 0
for value in [2, 3, 5]:
    total += value
print(total)
```

**What is printed?**

A. 5
B. 8
C. 10
D. 15

> Correct Answer: C. 10
> Overall Feedback: The loop accumulates 2 + 3 + 5, so the final output is 10.
